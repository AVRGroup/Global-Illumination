Crie a textura com ( necessario que o contexto do opengl já tenha sido criado ): 
Texture text = Texture("caminhoParaAtextura.jpg", TextureFormat)
TextureFormat :
TextureFormatRGB e TextureFormatRGBA para arquivos normais(jpg, tiff, png), (use TextureFormatRGBA apenas quando a imagem tiver um canal alpha)
TextureFormatRGB32F e TextureFormatRGBA32F para imagens em HDR, (use TextureFormatRGBA32F apenas quando a imagem tiver um canal alpha)
Os outros formatos ainda não foram completamente testados e são usados para rendertargets.


Para usar a textura no shader use o comando:
text.Use(textureUnit);

e para liberar a textura depois de usada
text.Release();

textureUnit deve ser o numero da unidade de textura que será utilizada, o mesmo numero que passa para o shader
glUniform1i(enderecoMapaDeTextura;, textureUnit);


####
# Shader
##########

uniform sampler2D MapaDeTextura;

in vec2 TexCoord0;
out vec4 color;

void main()
{
    vec4 color = texture(MapaDeTextura, TexCoord0);
}
